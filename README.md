# Retirement-Performance-02
Quick check of your retirement precentage gains

1) Put it on your phone & run it
iPhone (easy)

Save the folder somewhere you can access (iCloud Drive / Files).

Tap index.html → open in Safari.

Tap Share → Add to Home Screen (optional).

Android (easy)

Copy the folder to your phone (Downloads or any folder).

Open a file manager → tap index.html.

Choose Chrome (or your browser).

In Chrome menu → Add to Home screen (optional).

2) How to use it 

Pick Start Date and End Date

Enter Beginning Balance at the start date

Enter Ending Balance at the end date
It will show:

Simple Gain % (pure Begin → End)

Flow-adjusted Gain % (if you add contributions/withdrawals)

CAGR annualized (optional)
